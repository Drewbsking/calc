AASHTO Vertical Crest Curve

Purpose: Use this calculator to estimate the minimum required length and K value for a crest vertical curve connecting two vertical tangents, based on the selected stopping or passing sight-distance criterion.

How to use: Select AASHTO Stopping, RCOC Stopping, or AASHTO Passing. The calculator applies the matching sight-distance table and default heights automatically. Then enter the initial and final tangent grades plus the design speed. If needed, you can still edit the eye and object heights manually for a custom check.
